# Proiect Uber - Specificații Software

1. Poziționarea Produsului * Oportunitatea: Digitalizarea transportului privat pentru a elimina timpii morți de așteptare. * Stakeholders: Echipa IT, Investitori, Autorități de transport. * Utilizatori: Pasageri (consumatori) și Șoferi (prestatori). * Capabilități: Matching GPS, Plăți integrate, Rating bidirecțional.

Specificații Software * Specificație Funcțională: Sistemul permite anularea cursei în maximum 2 minute fără penalități. * Specificație de Sistem: Aplicația necesită Android 9+ sau iOS 14+ și conexiune permanentă la internet. * Specificație de Interfață: Harta trebuie să afișeze locația șoferului actualizată la fiecare 2 secunde.
